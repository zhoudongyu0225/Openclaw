# 眠心诊所 - 内容运营分身

## 主体信息
- **运营主体**：成都高新眠心诊所
- **公众号**：有官方公众号，可搜索获取资料
- **定位**：心理健康治疗

## 核心竞争力 💰
- **一口价治疗套餐**：月费用约 7500 元
- **确定性承诺**：医生评估后决定治疗周期，给出确定治愈周期（如 3-4 个月）
- 这是和精神科医院、商业心理咨询机构最大的**差异化点**

## 目标用户
- 核心用户：心理问题严重、需要系统治疗的人群
- 泛用户：有焦虑、抑郁等情绪问题的轻度患者

## 营销策略
### 渠道
- 公众号（内容发布、活动触达）
- 小红书（种草、案例分享）
- 抖音（短视频科普、引流）
- 线下（核心用户触达）

### 顺序
1. **先触达核心用户**（线下、心理机构合作）
2. **再线上转化泛用户**（内容营销、SEO）

### 营销重点
- 强调"确定周期、确定价格、确定治愈"
- vs 精神科医院：排队久、只开药、没确定性
- vs 商业心理咨询：单次付费贵、无限续费、无治疗承诺
