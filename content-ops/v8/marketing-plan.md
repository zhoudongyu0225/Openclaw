# 🏥 眠心诊所 - 营销计划

## 一、现状分析

### 业务定位
- 名称：眠心诊所
- 位置：成都
- 主营业务：心理咨询、睡眠障碍治疗

### 目标用户
- 25-45岁职场人群
- 有失眠、焦虑、压力问题
- 有付费能力

---

## 二、营销策略

### 1. 公众号内容矩阵

**内容类型：**
| 类型 | 频率 | 目的 |
|------|------|------|
| 睡眠科普 | 3篇/周 | 建立专业形象 |
| 案例故事 | 1篇/周 | 引发共鸣 |
| 互动话题 | 1篇/周 | 提高活跃度 |
| 优惠活动 | 1次/周 | 转化 |

**爆款标题公式：**
- "XX竟然是因为XX，看完你就懂了"
- "3个方法让你XX，亲测有效"
- "为什么XX的人更容易XX"

### 2. 私域运营

**引流路径：**
```
公众号 → 添加微信 → 进入社群 → 1v1咨询
```

**社群运营：**
- 每日睡眠小知识
- 每周专家问答
- 每月主题活动

### 3. 短视频（抖音）

**内容方向：**
- 科普类：30秒讲清楚一个睡眠问题
- 案例类：真实咨询故事（经授权）
- 互动类：测试你的睡眠质量

**发布时间：** 早7-9点，晚7-10点

### 4. 小程序

**功能：**
- 在线预约
- 睡眠测评
- 知识库
- 会员积分
