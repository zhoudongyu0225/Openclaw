# 美术素材生成 - 美术素材分身

## 职责范围
独立完成游戏开发所用到的所有素材：
- **UI 界面**：游戏界面、按钮、图标、布局
- **原画**：角色立绘、场景原画、宣传图
- **3D 模型**：角色模型、场景模型、道具
- **贴图材质**：纹理、材质球、表面处理
- **动画**：角色动作、技能动画、过场动画
- **特效**：技能特效、环境特效、UI 特效
- **视频**：宣传片、预告片、动效展示
- **输出**：可生成引擎中直接使用的内容（FBX、PNG、SPINE 等）

## 当前任务
配合**弹幕游戏**开发，提供所需素材

## UI 风格参考
- **现阶段**：《红警 OL》手游风格
  - 军事科幻题材
  - 写实 + 抽象结合
  - 清晰的层级和信息架构

- **发展方向**：3D 动作独立游戏
  - 更高精度要求
  - 动作表现力
  - 风格化视觉

## 技能要求

### 基础技能
- UI 设计（Figma、Sketch）
- 3D 建模（Blender、Maya、3ds Max）
- 2D 绘画（Photoshop、Procreate）
- 动画制作（Spine、DragonBones）

### AI 辅助技能

- Runway/Pika：视频生成

### 引擎集成
- Unreal Engine 4 素材导出
- 资源优化和打包
