# 素材制作工作流

## 一、需求确认

### 1.1 接收需求
- 明确素材用途（UI/模型/特效）
- 确认风格参考
- 确定交付时间

### 1.2 需求评审
- 技术可行性评估
- 资源评估
- 制定计划

---

## 二、概念设计

### 2.1 AI生成概念图
```
工具：Midjourney / Nano Banana 2
输入：提示词 + 风格参考
输出：3-5个概念方案
```

### 2.2 方案确认
- 内部评审
- 客户/负责人确认
- 锁定最终方案

---

## 三、原型制作

### 3.1 UI制作
```
工具：Figma / Photoshop
输出：分层设计稿
```

### 3.2 3D建模
```
工具：Blender
流程：建模 → 拓扑 → UV → 材质 → 渲染
```

### 3.3 动画制作
```
工具：Spine / DragonBones / UE4
流程：骨骼 → 蒙皮 → 关键帧 → 动画
```

---

## 四、引擎集成

### 4.1 UE4导入
- FBX/PNG导入
- 材质设置
- 蓝图配置

### 4.2 性能优化
- Draw Call优化
- 纹理压缩
- LOD设置

---

## 五、测试验收

### 5.1 功能测试
- UI交互正常
- 模型显示正确
- 动画播放流畅

### 5.2 性能测试
- 帧率达标
- 内存占用合理
- 加载时间合格

---

## 六、交付归档

### 6.1 交付
- 整理文件结构
- 编写说明文档
- 提交验收

### 6.2 归档
- 备份源文件
- 更新素材库
- 记录经验教训

---
