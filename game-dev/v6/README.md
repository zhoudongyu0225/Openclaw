# 弹幕游戏开发 - 游戏开发分身

## 项目概述
- **类型**：抖音/快手/tiktok 弹幕小游戏
- **题材**：恐龙题材攻守塔防
- **核心玩法**：双主播直播间 PK

## 玩法详解 🦖

### 双方机制
- 红色方（攻击方）：主播直播间看到的敌方，生成各种恐龙兵种进攻
- 蓝色方（防御方）：我方防御塔和基地

### 主播操作
- **防御视角**：能看到我方防御塔和基地（蓝色），指挥防御敌方恐龙
- **进攻视角**：能看到敌方防御塔和基地，指挥进攻

### 胜负条件
- 任意一方基地被敌方刷出的恐龙破坏即失败
- 类似于"恐龙版 IO 游戏"

## 后端系统架构

### 核心模块
1. **主播 PK 系统**
   - 匹配对手
   - 实时同步状态
   - 胜负判定

2. **平台 SDK 对接**
   - 抖音小程序 SDK
   - 快手小游戏 SDK
   - 礼物系统对接

3. **API 开发**
   - 用户认证
   - 房间管理
   - 战绩查询

4. **通信传输**
   - WebSocket 实时通信
   - Protobuf 序列化

5. **主播管理**
   - 主播入驻
   - 数据统计
   - 收益分成

6. **直播间玩家管理**
   - 观众互动
   - 弹幕抽奖
   - 观战系统

7. **GM 后台**
   - 房间管理
   - 参数配置
   - 异常监控

8. **模拟礼物工具**
   - 测试用
   - 压力测试

## 技术栈
- **游戏引擎**：Unreal Engine 4
- **后端**：Go
- **数据库**：MongoDB + Redis
- **协议**：Protobuf
- **平台**：抖音、快手（国内）→ tiktok（海外）

## 多语言适配
- 国内：中文
- 海外：英文、日文、韩文等多语言
