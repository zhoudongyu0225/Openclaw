# 快手弹幕玩法 - 技术选型指引

## 三种开发方案对比

| 对比项 | PC伴侣 | 云平台 | 小程序 |
|--------|--------|--------|--------|
| **设备要求** | 无 | 无（需安卓2000+/苹果10+） | 无 |
| **视觉效果** | 强 | 强（不受机型影响） | 受JS引擎限制 |
| **战斗规模** | CP自己约束 | 50人同时下场 | CP自己约束 |
| **操作延迟** | 低 | 50ms+链路 | 无延时 |
| **网络带宽** | 无要求 | 需8M拉流 | 无要求 |

## 开发约束

### PC伴侣/云平台
- 引擎：Cocos、UE、U3D均可，打exe包
- 包体积：5G以内

### 小程序
- 引擎：Unity、Cocos、Laya、白鹭、Three.js
- 包体积：Cocos首包4MB/总20MB，Unity 100MB
- 网络：Websocket、XMLHttpRequest

## 性能要求

| 类型 | 要求 |
|------|------|
| 渲染帧率 | >= 30 fps |
| 逻辑帧率 | >= 15 fps |
| 内存 | <= 16G |
| CPU | 12400f |
| 显卡 | 3060ti |

## 适合玩法

| 方案 | 适合类型 |
|------|----------|
| PC伴侣 | SLG、RPG、MMO（复杂特效） |
| 云平台 | 动作类、Rogue、竞速、射击 |
| 小程序 | 轻量小游戏 |
