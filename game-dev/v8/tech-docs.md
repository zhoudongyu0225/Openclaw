## 快手弹幕玩法技术接口

### 基础能力（上线必须）
1. **主播链路** - 绑定相关
2. **对局相关** - 置顶礼物
3. **推送互动消息** - 点赞、评论、送礼、关注、分享、粉丝团、超粉团、贵族
4. **用户头像处理** - 图片缩放
5. **消息回查** - 加固能力

### 产品扩展能力
1. **用户链路** - 快捷评论、加入战队
2. **礼物效果** - 礼物效果提示
3. **弹幕底部栏** - 小程序&内购礼物
4. **主播链路** - 一键开播、连屏
5. **可操作** - 摇杆云、小程序可操作

### 通用能力
1. **用户反查** - 用户id→openId
2. **用户信息** - 粉丝团、贵族、等级、排名
3. **云开播** - 移动端/PC
4. **云支付** - 玩法内送礼
5. **组队语音** - 频道控制、麦位管理
6. **游戏帧数据同步** - 推送给观战用户
7. **压测** - 点赞、送礼、评论

---

## 性能要求

### 压测标准
- 单玩法：评论600qps、点赞3000qps、送礼600qps
- 单直播间：max 1000 qps
- 全局：4200 qps
- 客户端：10分钟不崩溃，fps≥18
- 对战类：支持单场5w UV
- 推送接口P995<1000ms

### 监控要求
- 主播在线数、掉量报警
- 推送数据同环比报警
- 基础组件可用性报警

---

## 上线规范

### 时间
- 周一~周四：10-12点、14-18点
- 周五：尽量不上
- 周末：禁止上线

### 要求
- 可监控
- 可灰度放量（1%→10%→30%→60%→100%）
- 可回滚
- 禁止热更新绕过审核！

### 通报
- 10点~22点，每4小时巡检
