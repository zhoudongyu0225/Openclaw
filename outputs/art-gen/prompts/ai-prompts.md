## 8. ⚙️ 常用参数速查

| 参数 | 说明 | 常用值 |
|------|------|--------|
| --ar | 宽高比 | 16:9, 4:3, 1:1, 3:2 |
| --v | 版本 | 6 (最新) |
| --s | stylize风格化 | 100-1000 |
| --sref | 风格参考 | style reference |
| --no | 排除元素 | --no text, --no watermark |
| --iw | 图像权重 | 0-1 |

## 9. 使用技巧

1. **颜色代码**：使用 #HEX 代码精确控制颜色
2. **主题词放前面**：重要描述放在提示词开头
3. **指定渲染引擎**：unreal engine, blender, octane
4. **添加stylize**：--s 150-300 提升艺术效果
5. **透明背景**：最后加 transparent background
6. **游戏资产**：加 game asset, white background, clean render
7. **UI设计**：加 ui ux design, figma style, mockup

## 18. 📊 提示词性能优化技巧

### 提高生成质量
1. **添加参考引擎**：unreal engine 5, blender cycles, octane render
2. **使用stylize**：--s 150-300 增加艺术性
3. **指定光照**：cinematic lighting, studio lighting, volumetric
4. **添加细节词**：highly detailed, intricate, elaborate

### 避免常见问题
1. **排除不需要的**：--no text, --no watermark, --no ui
2. **明确背景**：white background, transparent, solid color
3. **指定视角**：three-quarter view, hero shot, top-down
4. **控制复杂度**：low-poly, simplified, stylized

### 游戏资产专用
- 模型：game asset, white background, clean render, game-ready
- UI：ui ux design, mockup, figma style, professional layout
- 特效：sprite sheet, particle system, transparent PNG
- 2D：character design, concept art, card art
