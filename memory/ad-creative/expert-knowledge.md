- **目标用户**：25-40 岁，RPG 爱好者
- **买量重点**：

## 创意模板库

### 模板 1：真假主角
```
[0-3s] 展示一个"loser"角色被嘲笑
[3-8s] 意外获得游戏道具/武器，战斗力爆增
[8-10s] 逆转打脸，之前嘲笑的人傻眼
[10-12s] 游戏画面 + 下载按钮
```

## 平台投放策略

### 抖音
- **视频时长**：15-30 秒
- **前 3 秒**：必须有钩子（冲突/悬念/爽点）
- **字幕**：大字幕，关键信息突出
- **BGM**：节奏感强，热点音乐
- **投放时段**：12-14点、18-22点
- **出价**：CPM 15-30 元

### 快手
- **视频时长**：15-45 秒
- **风格**：更接地气，老铁文化
- **投放时段**：12-14点、19-23点
- **出价**：CPM 10-20 元

### 海外 TikTok
- **视频时长**：15-30 秒
- **本地化**：英语/日语/韩语配音
- **文化差异**：避免敏感话题
- **投放时段**：根据目标国家时区

## 投放数据分析

### 核心指标
| 指标 | 含义 | 合格线 |
|------|------|--------|
| CTR | 点击率 | 2%+ |
| CVR | 转化率 | 1%+ |
| CPC | 单次点击成本 | < 1 元 |
| CPA | 单次转化成本 | < 30 元 |
| 完播率 | 看完比例 | 30%+ |
| 3 秒播放率 | 看 3 秒比例 | 50%+ |

### 优化思路
```
CTR 低 → 换开头钩子
CVR 低 → 优化落地页/转化路径
CPA 高 → 优化人群定向
完播率低 → 缩短视频/加快节奏
```

## 广点通/巨量引擎投放 SOP

### 1. 创建计划
- 选择投放目的：应用下载
- 选择对应应用
- 设置预算（日预算 1000-5000 元）

### 2. 人群定向
- 年龄：25-45 岁
- 性别：男
- 兴趣：游戏、军事、历史
- 行为：游戏下载用户

### 3. 出价策略
- 优先放量模式
- 出价 = 目标 CPA × 1.2

### 4. 素材选择
- 准备 3-5 套素材
- 每套素材不同开头
- AB 测试优化

### 5. 数据监控
- 每 2 小时看一次数据
- CTR < 1% 换素材
- 成本太高调整出价
- 量级稳定后拓量

## 常见问题

### Q: 素材做出来了但不过审？
A: 检查是否有：
- 绝对化用词（第一、最好）
- 虚假宣传（轻松赚钱）
- 敏感内容（政治、色情）
- 侵权素材（其他游戏画面）

### Q: 投放成本太高怎么办？
A:
1. 优化落地页加载速度
2. 调整人群定向
3. 换新素材
4. 尝试新渠道

### Q: 量级太小怎么办？
A:
1. 放宽定向
2. 提高出价
3. 增加素材
4. 尝试新计划
