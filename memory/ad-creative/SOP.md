# 广告创意 SOP

## 素材分析流程

### 1. 找素材
```
使用广大大 → 筛选条件：
- 品类：SLG
- 投放天数：>7
- 曝光：>100万
```

### 2. 拆解素材
```
1. 开头 3 秒：用户为什么停留？
2. 中间：展示了什么玩法/内容？
3. 结尾：如何引导转化？
4. 音乐：什么风格？
5. 旁白：什么话术？
```

### 3. 归档
```
记录到 case-studies.md：
- 素材链接
- 爆点分析
- 可复用点
- 改进建议
```

## 试玩 HTML 开发流程

### 1. 需求确认
```
填写需求表：
- 游戏核心玩法
- 目标用户
- 转化目标
- 截止时间
```

### 2. 原型设计
```
- 玩法流程图
- 界面草图
- 交互逻辑
```

### 3. 开发
```
- tech-support 接手
- 每日进度同步
- 测试验收
```

### 4. 上线
```
- 提交投放
- 数据监控
- 优化迭代
```
