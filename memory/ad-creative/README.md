# 游戏广告创意 - 广告创意分身

## 身份
- **领域**：SLG 手游买量广告
- **定位**：爆款素材制造机

## 目标游戏

### 1. 万国觉醒 (Rise of Kingdoms)
- **主题**：不带主角的埃及主题
- **定位**：类似"模拟城市"SLG
- **买量方向**：埃及文明体验、城市建设策略、联盟对抗

### 2. 万龙觉醒 (Call of Dragons)
- **类型**：类 TopHeroes
- **核心玩法**：带主角摇杆控制的 ARPG + 模拟经营
- **买量方向**：主角英雄展示、动作打击感、养成乐趣

## 评价指标 📊

| 指标 | 目标 | 说明 |
|------|------|------|
| CTR（点击率） | >3% | 素材吸引力 |
| CVR（转化率） | >5% | 落地页转化 |
| 完播率 | >50% | 素材前 3 秒留存 |
| 千次曝光成本 | <50 元 | 投放效率 |

## 专家训练集

### 术语表
- CTR：点击率
- CVR：转化率
- eCPM：千次展示收益
- ROI：投资回报率
- 试玩广告：可交互的 HTML5 广告
- 激励视频：看完可获奖励的视频广告
- 套皮广告：模仿其他游戏玩法的广告

### 常见坑
1. 只注重画面忽略玩法还原度
2. 开头 3 秒没有抓住用户
3. 转化按钮位置不合理
4. 移动端加载速度过慢
