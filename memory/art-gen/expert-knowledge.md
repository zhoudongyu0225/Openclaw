## 视觉风格参考

### 弹幕游戏 UI 风格
- **参考对象**：《红警 OL》手游
- **特点**：
  - 厚重科技感
  - 明确功能分层
  - 强按钮反馈
  - 信息密度高但不杂乱

### 配色方案
```
主色：#D8B36A (金色，尊贵感)
辅色：#8FA8C9 (蓝灰，科技感)
背景：#1A1F2E (深蓝黑)
强调：#FF6B6B (红色，警示/攻击)
成功：#4CAF50 (绿色，确认)
```

### 字体
- **中文**：思源黑体 Bold
- **英文/数字**：Roboto Mono
- **标题**：48-64px
- **正文**：24-32px

## 产出规范

### UI 组件库

#### 按钮
```
尺寸：
- 大：200x60px
- 中：160x50px
- 小：120x40px

样式：
- 背景：渐变 + 边框发光
- 状态：Normal / Hover(更亮) / Pressed(内凹) / Disabled(灰)
- 点击反馈：0.1s 缩放 + 粒子特效
```

#### 卡片
```
圆角：16px
阴影：0 4px 20px rgba(0,0,0,0.3)
边框：1px rgba(255,255,255,0.1)
内边距：20px
```

#### 进度条
```
高度：20px
背景：rgba(255,255,255,0.1)
填充：渐变色（绿→黄→红）
动画：平滑过渡
```

### 恐龙兵种设计

#### 分类
| 类型 | 特点 | 示例 |
|------|------|------|
| 突击型 | 速度快、血少 | 迅猛龙 |
| 坦克型 | 速度慢、血厚 | 霸王龙 |
| 飞行型 | 绕防御塔 | 翼龙 |
| 群居型 | 小恐龙成群 | 伤齿龙 |

#### 视觉规范
- 风格：低多边形 + 细节纹理
- 颜色：红色系（敌方）vs 蓝色系（我方）
- 尺寸：塔防视角要清晰可辨
- 动画：行走、攻击、受击、死亡

### 防御塔设计

#### 分类
| 类型 | 特点 | 示例 |
|------|------|------|
| 炮台 | 高伤害、单体 | 激光塔 |
| 范围 | AOE 伤害 | 导弹塔 |

## 3D 发展方向

### 3D 建模规范

#### 软件
- **建模**：Blender
- **雕刻**：ZBrush
- **纹理**：Substance Painter
- **引擎**：UE4/5

#### 建模流程
```
1. 概念设计（2D）
2. 中模制作（Blender）
3. 高模雕刻（ZBrush）
4. 拓扑低模
5. UV 展开
6. 贴图绘制
7. 引擎导入测试
8. 动画绑定
```

### 动画制作
