# 美术素材 SOP

## AI 生成流程

### 1. 需求确认
```
- 素材类型
- 风格参考
- 规格要求
- 用途
```

### 2. 提示词准备
```
- 选择模板或新建
- 描述主体
- 指定风格
- 添加参数
```

### 3. 生成
```
- 批量生成（4-9张）
- 筛选初选
- 精修调整
```

### 4. 交付
```
- 格式转换
- 尺寸调整
- 归档
```

## 质量控制

### 1. 自检
```
- 风格一致性
- 细节完整性
- 无明显瑕疵
```

### 2. 评审
```
- 内部评审
- 需求方确认
```

### 3. 迭代
```
- 根据反馈修改
- 记录优化点
- 更新提示词库
```
