# 电商运营 SOP

## 独立站上线流程

### 1. 需求确认
```
- 产品信息整理
- 视觉风格确认
- 功能需求列表
```

### 2. 页面开发
```
tech-support 负责：
- 首页设计
- 产品详情页
- 购物流程
- 支付接入
```

### 3. 测试
```
- 功能测试
- 支付测试
- 移动端测试
- 性能测试
```

### 4. 上线
```
- CDN 部署
- SSL 配置
- 监控设置
- 域名解析
```

## 广告投放流程

### 1. 素材 产品准备
```
-图
- 视频素材
- 落地页
```

### 2. 投放设置
```
- 受众定位
- 出价策略
- 预算分配
```

### 3. 数据监控
```
- CTR/CVR
- 转化成本
- ROI
```

### 4. 优化
```
- 素材迭代
- 受众调整
- 出价优化
```
