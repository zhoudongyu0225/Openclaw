# V113：5分钟急救入睡（生活妙招型）

【画面】
深夜，主人公躺在床上，盯着天花板，烦躁地翻身。

【配音】
"又失眠了？试试这个方法，5分钟就能睡着。"

【画面】
字幕出现数字倒计时，配合动作演示。

【配音】
"第一步，渐进式肌肉放松。从脚趾开始，用力绷紧5秒，然后突然放松。脚、小腿、大腿...一直往上。"

【画面】
主人公做放松动作，逐渐安静下来。

【配音】
"第二步，4-7-8呼吸。吸气4秒，屏住7秒，呼气8秒。重复3次。"

【画面】
主人公呼吸逐渐平稳。

【配音】
"第三步，想一个让你安心的画面。阳光、海滩，或者猫咪在怀里打呼噜。"

【画面】
主人公表情放松，慢慢闭上眼睛。

【字幕】
"5分钟急救入睡法，转发给那个失眠的朋友"

【结束】
*眠心诊所*
