# V128：全身酸痛原因（问题解决型）

## 【开场】
"一觉醒来全身酸痛？可能不是没睡好，是这个问题"

## 【正文】
"是不是经常这样：睡了一晚上，早上起来比没睡还累？

原因可能是这4个：
第一，床垫太旧了。超过8年的床垫支撑力下降，身体承受额外压力。

第二，睡姿不对。俯卧会让颈椎腰椎一晚上都在受力。

第三，室温太低。冬天睡觉肌肉僵硬，醒来全身酸胀。

第四，睡前没拉伸。运动后肌肉处于紧绷状态。

解决方法：
1. 5-8年换一次床垫
2. 侧卧时枕头跟肩一样高
3. 室温调到20-24度
4. 睡前做10分钟拉伸"

## 【结尾】
"收藏起来自查一下"
