# 抖音脚本 V13：失眠患者的一天（30秒）

## 脚本时长：30秒

### [0-5s] 场景代入
画面：昏暗房间，时钟指向凌晨3点
配音：我以为失眠只是睡不着，其实它毁掉的是一整天。

### [5-15s] 早上
画面：早上闹钟响，人很疲惫
配音：早上7点，闹钟响了。
昨晚只睡了3小时，头晕、乏力、反应迟钝。
灌了两杯咖啡，才能出门上班。

### [15-22s] 下午
画面：办公室，人很困但睡不着
配音：下午3点，状态最差。
领导说的话转头就忘，
同事问我在想什么，我说自己也不知道。

### [22-28s] 晚上
画面：晚上躺在床上，睁着眼睛
配音：晚上11点，开始恐惧。
"今晚能睡着吗？"
越想越清醒，凌晨3点，又醒了。

### [28-30s] 转折
画面：眠心诊所 Logo
配音：失眠6年，我终于治好了。
如果你也在经历，回复"失眠"，医生帮你。

---

## 备注
- 语速：慢，沉重
- 画面：昏暗、压抑
- 配乐：低沉钢琴
- 目标：引发共鸣，引导私信
