# 公众号文章81：睡前1小时做什么，决定你能不能睡着

很多人问：为什么我明明很困，躺下却睡不着？

答案很可能藏在**睡前1小时**。

## 睡前1小时的"死亡组合"

- 刷短视频 → 大脑兴奋
- 看新闻/热点 → 焦虑激活
- 躺床上玩手机 → 手机蓝光+思维活跃

这三个习惯，几乎是失眠的"标准配置"。

## 正确的睡前1小时应该做什么？

### 第30分钟：进入"低刺激模式"

- 关掉短视频、新闻、社交媒体
- 把灯光调暗（不是全黑，是变暗）
- 开始做一些**不需要思考**的事

### 第20分钟：放松身体

- 简单的拉伸或冥想
- 温水澡（不是热水，会反而兴奋）
- 深呼吸（4-7-8呼吸法）

### 第10分钟：创造睡眠环境

- 卧室温度调到16-19℃
- 关窗隔音
- 躺在床上，做"身体扫描"——从脚趾到头顶，一个部位一个部位地放松

## 关键：睡前1小时不要做决定

研究发现，睡前做决定的人，第二天的焦虑水平显著更高。

**把"想事情"这件事，留到白天。**

---

好睡眠不是"困了就能睡"。

而是你给大脑足够的信号："今晚的工作结束了。"

转发给那个失眠的朋友。
