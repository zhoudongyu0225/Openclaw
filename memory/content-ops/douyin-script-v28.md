# 抖音脚本 V28：你的睡姿正在毁掉你的睡眠

**类型**：痛点共鸣型  
**时长**：30-45秒  
**目标人群**：20-50岁有睡眠问题的人群

---

## 开头（3秒）
画面：昏暗房间，各种奇葩睡姿快速切换
台词：/"仰着睡、侧着睡、趴着睡...你知道吗，你的睡姿可能正在毁掉你的睡眠！/

## 干货内容（30秒）

### 1. 趴着睡（8秒）
画面：俯卧姿势，配"X"符号
台词：/"第一个，趴着睡。绝对禁止！压迫内脏，压迫颈椎，还容易落枕。/

### 2. 右侧卧（8秒）
画面：右侧卧姿势，配警告符号
台词：/"第二个，右侧卧。长期这样睡，腰椎间盘突出了解一下。/

### 3. 正确睡姿（14秒）
画面：仰卧+左侧卧正确示范
台词：/"正确的睡姿是仰卧，或者左侧卧。仰卧时膝下垫个枕头，侧卧时两腿夹个枕头。记住：枕头要支撑颈椎，不是只枕头。很简单，照做就行。/


## 结尾（5秒）
画面：Easy face + 诊所logo
台词：/"睡姿对，事半功倍。关注我，教你睡个好觉。/


## 字幕设计
- "绝对禁止"加红
- "腰椎间盘突出"放大
- 正确睡姿部分逐步展示
- 配乐：节奏感强的卡点音乐

## 拍摄建议
- 开头用快速切换制造视觉冲击
- 讲解时语气肯定，有说服力
- 可以用动画或示意图辅助说明
