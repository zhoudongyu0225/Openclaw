# 公众号文章85：为什么记忆里越来越差？可能是你睡得不对

刚说完的话转头就忘？

东西放在哪里完全记不住？

别人名字叫不出来？

别急着怪脑子，可能是**你睡错了**。

## 睡眠对记忆有多重要？

### 记忆巩固发生在睡眠中

白天学到的东西，需要在睡眠中"存档"。

- **海马体**：白天收集信息
- **大脑皮层**：睡眠时整合储存

睡不好 = 硬盘没保存。

### REM睡眠决定程序性记忆

- 骑车、弹琴、游泳……这些技能记忆
- 需要在REM睡眠（快速眼动期）中被强化

### 深度睡眠决定陈述性记忆

- 单词、数字、人名……这些知识记忆
- 需要在深度睡眠中被巩固

## 你的记忆问题出在哪？

| 症状 | 可能原因 |
|---|---|
| 记不住新东西 | 深度睡眠不足 |
| 技能退步 | REM睡眠不足 |
| 容易走神 | 整体睡眠质量差 |

## 怎么睡才能记性好？

1. **睡够7-8小时**
2. **固定起床时间**
3. **睡前1小时不学习**
4. **午睡20分钟**（补充分类记忆）

## 一个真相

你熬的每一次夜，都是在消耗记忆力。

那些说"我天生记性差"的人，大部分只是睡得不对。
