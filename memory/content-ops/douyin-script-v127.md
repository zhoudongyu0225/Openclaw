# V127：癌症信号出现在睡觉时（痛点共鸣型）

## 【开场】
（画面：深夜开着灯醒来的场景）
"睡觉时出现这几种症状，可能是身体在报警"

## 【正文】
"很多人忽视了这个：
第一个，夜间盗汗。不是一般的出汗，是那种把被子都浸湿的汗，可能是淋巴瘤、白血病的信号。

第二个，不明原因体重下降＋失眠。很多癌症早期都会出现这个组合。

第三个，持续性骨痛。晚上睡觉时骨头疼得厉害，不是着凉那么简单。

第四个，反复发烧。找不到原因的发热，尤其在夜间加重。

当然，有这些症状不代表一定是癌症，但持续出现的话，建议去体检一下。"

## 【结尾】
"转发给身边爱的人，身体发出的信号要重视。"
