# 抖音脚本V43：睡眠呼吸暂停自测

## 类型：健康科普型
## 时长：30秒

【画面】
开场：男主躺在床上打呼噜，突然屏住呼吸
字幕：打—停—喘气

【配音】
"打呼噜的小伙伴注意了！你以为这是睡得香？其实是身体在求救！"

【画面】
切到科普卡片
字幕：三个信号

【配音】
"第一，呼噜打着打着突然停了，超过10秒；第二，早上起来头痛、口干；第三，白天犯困，开车都能睡着。"

【画面】
男主开车，不断点头犯困

【配音】
"有这三个情况，可能是睡眠呼吸暂停综合症。"

【画面】
出现建议
字幕：侧卧睡、减肥、戒烟酒

【配音】
"轻症的先侧卧睡、减减肥，严重的一定要去医院！"

【结尾】
字幕：转发提醒你的呼噜王朋友
---

*备注：语速偏快，情绪有起伏，最后一句带有社交属性*
