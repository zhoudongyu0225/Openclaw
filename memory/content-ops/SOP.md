# 内容运营 SOP

## 公众号发布流程

### 1. 选题
```
来源：
- 竞品爆款
- 用户反馈
- 行业热点
- 案例库
```

### 2. 撰写
```
结构：
- 标题（引发好奇）
- 引入（场景切入）
- 干货（解决问题）
- 案例（增强信任）
- 转化（引导私私信）
```

### 3. 排版
```
- 字号：15px
- 配色：品牌色
- 配图：相关且高清
- 长度：1500-2000字
```

### 4. 发布
```
- 预览确认
- 定时发送
- 朋友圈转发
- 数据监控
```

## 私私信转化流程

### 1. 响应
```
- 30分钟内回复
- 态度专业友好
```

### 2. 了解需求
```
- 什么问题？
- 严重程度？
- 治疗意愿？
```

### 3. 引导留资
```
- 介绍一口价套餐
- 邀请到院评估
- 记录联系方式
```

### 4. 跟进
```
- 当日回访
- 3日内再次跟进
- 记录结果
```
