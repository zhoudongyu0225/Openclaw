# V41：睡前情绪不好可能是吃错了

## 类型：问题解决型
## 时长：30-45秒

### 画面设计

**开场（0-3秒）**
画面：深夜，一个人躺在床上翻来覆去，面露焦虑
配音：很多人睡前突然情绪低落，却不知道原因

**第二幕（3-15秒）**
画面：快切各种食物：奶茶、蛋糕、辣条、炸鸡
配音：你以为是自己想太多？其实是吃错了！

**干货（15-30秒）**
画面：字幕列出三类食物
配音：睡前不要吃：高糖零食——血糖波动引发焦虑；油炸食品——加重身体负担容易做噩梦；咖啡因——半衰期6小时，下午喝晚上也清醒

**结尾（30-45秒）**
画面：换成吃香蕉、喝牛奶
配音：改成吃这些：香蕉补镁放松神经，温牛奶助眠又安神。记住，问题不在心里，在胃里！

### 字幕要点
- "睡前emo"用引号突出
- 三类食物用醒目颜色标注
- 结尾"问题不在心里，在胃里"放大加粗

### 音乐
轻快的卡点音乐，节奏感强
