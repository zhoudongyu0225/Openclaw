# 抖音脚本 V91：鬼压床的真相

## 类型：知识科普型

## 脚本

【画面：有人躺床上，挣扎着醒不来】

你有没有过这种经历？躺下睡觉，突然醒了，身体却动不了，想喊救命喊不出来，感觉胸口被压着——这就是"鬼压床"。

别怕，今天告诉你真相。

【动画演示大脑和身体的"通信故障"】

其实这是大脑和身体没同步好。大脑醒了，身体还在睡觉，两边没对上号。

【列出谁更容易遇到】

这几种人最容易"鬼压床"：
① 经常熬夜、睡眠不足的
② 压力大、焦虑的
③ 作息不规律的
④ 喜欢仰卧睡的

【快速解决tips】

遇到怎么办？记住3个字：
- 放松（别慌，一慌更醒不过来）
- 动指（先动手指、脚趾，慢慢唤醒身体）
- 眨眼（快速转动眼球，能加速醒来）

【提醒】

偶尔一次不用管，但如果频繁出现，一周好几次，去医院查查，可能是其他睡眠问题。

我是眠心，帮你睡个好觉。
