# 抖音脚本 V17：测测你的焦虑程度

## 互动测试型

### 开头（0-3秒）
画面：紧张的音乐，一个焦虑表情包
文案：焦虑自测！你在第几级？

### 内容（20-30秒）
1. **轻度焦虑**
   "偶尔担心，心跳加快，但能控制"
   （状态：紧张但不影响生活）
   
2. **中度焦虑**
   "经常担心，容易失眠，影响工作"
   （状态：无法集中注意力）
   
3. **重度焦虑**
   "过度担忧，无法出门，心慌手抖"
   （状态：严重影响日常生活）
   
4. **焦虑症**
   "持续紧张6个月以上，有躯体症状"
   （状态：需要专业治疗）

### 结尾（5秒）
画面：舒一口气的表情
文案：轻度可以自愈，中度以上建议就医
关注我，下期讲焦虑症怎么治

### 时长：约35秒
### 风格：互动测试 + 知识科普
### 配乐：轻快/悬疑

## 抖音脚本 V18：睡眠科普辟谣

## 知识辟谣型

### 误区一：喝红酒助眠
"红酒确实让人犯困，但会降低睡眠质量，
让你在浅睡眠里醒不来。"

### 误区二：睡不着就吃安眠药
"安眠药有依赖性，必须在医生指导下使用，
自己乱吃可能越吃越多。"

### 误区三：睡得越多越好
"成年人7-8小时足够，
睡太久反而头晕乏力。"

### 误区四：打呼噜=睡得香
"打呼噜可能是呼吸暂停综合征，
严重时会猝死。"

### 误区五：白天补觉
"白天睡再多也补不回晚上的觉，
还会打乱生物钟。"

### 结尾
关注眠心 clinic，
带你科学睡眠。

### 时长：约45秒
### 风格：知识辟谣
### 配乐：轻快
