# V115：冬天睡不够（知识科普型）

【画面】
冬天早晨，主人公裹着被子不想起床。

【配音】
"为什么冬天睡10小时还是困？不是你懒，是季节在作妖。"

【字幕】
"原因1：日照时间短"

【画面】
窗外灰蒙蒙的天。

【配音】
"阳光少 = 维生素D不足 = 疲劳。"

【字幕】
"原因2：新陈代谢变慢"

【画面】
手机显示"省电模式"。

【配音】
"身体自动开启省电模式，人也变得昏昏欲睡。"

【字幕】
"原因3：褪黑素紊乱"

【画面】
天黑了的窗外。

【配音】
"天黑得早，褪黑素提前分泌，作息全乱了。"

【字幕】
"应对方法"

【画面】
主人公早晨晒太阳、吃维生素D、跑步。

【配音】
"早起晒太阳，补维生素D，适当运动。"

【画面】
主人公精神百倍地起床。

【配音】
"冬天不是猫，不需要冬眠。起来嗨！"

【字幕】
"转发给那个冬天起不来的人"

【结束】
*眠心诊所*
