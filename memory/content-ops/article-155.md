# 文章155：睡前这个行为，会让失眠越来越严重

## 开篇引入
很多人失眠后，会采取一些"自认为有效"的方法来帮助睡眠，结果却适得其反。今天说说那些会让失眠越来越严重的行为。

## 越做越失眠的4个行为

### 1. 强迫自己入睡
睡不着的时候，很多人会告诉自己"赶紧睡"，结果越强迫越清醒。大脑会形成"床=焦虑"的反射，一上床就紧张。

**正确做法：**睡不着就起来，离开床，做点放松的事，等困了再回来。

### 2. 反复看时间
失眠时看时间会让焦虑升级。"已经凌晨2点了，怎么办""再睡不着明天就完了"——这些想法会刺激大脑皮层兴奋。

**正确做法：**把手机闹钟设定好就放在够不着的地方，不要看时间。

### 3. 白天补觉
昨晚没睡好，白天疯狂补觉。结果晚上更睡不着，形成恶性循环。

**正确做法：**即使很困，白天也不 要超过30分钟的午睡，且在下午3点前完成。

### 4. 喝酒助眠
很多人觉得睡前喝点酒容易入睡。确实能快速入睡，但会严重影响睡眠质量，后半夜容易醒来，醒来后再也睡不着。

**正确做法：**睡前4小时不要饮酒，可以用温牛奶、放松训练代替。

## 失眠的正确应对方式

**建立睡眠仪式：**
- 固定时间上床
- 睡前放松活动
- 卧室只用于睡觉

**认知调整：**
- 接受偶尔失眠是正常的
- 不要把失眠后果放大
- 相信身体有自我调节能力

## 结语
失眠本身不可怕，可怕的是对失眠的焦虑。很多时候，改变对失眠的态度，反而是最有效的治疗方法。
