# V33：周末补觉的正确方式

## 类型：知识干货型

## 脚本

【画面：周末睡到下午1点起床的人，显得很疲惫】

别再周末睡到下午了。

【画面：字幕浮现】

告诉你一个颠覆认知的研究：**周末补觉，越补越累。】

【画面：解释"社交时差"】

你想啊，周内你7点起床，周末睡到12点——

相当于周一到周五在北京，周末跑到东京。

这种时差，比跨国飞行还伤身体。

【画面：为什么会越睡越累】

而且睡太久会：
- 脑子发懵（睡眠惯性）
- 皮质醇升高（反而更累）
- 生物钟乱套（周一更起不来）

【画面：正确补觉方式】

那怎么补？

【画面：三种方法】

方法一：每天提前15分钟睡

方法二：周末睡到9点就行，下午小睡20分钟

方法三：困了随时小睡，但别超20分钟

【画面：总结】

记住一句话：**睡眠是每天的工程，别想周末一次性竣工。**

【画面：结尾】

我是眠心，关注我，每天睡个好觉。

## 时长：约50秒
## 核心知识点：社交时差、科学补觉
## 引导：关注+收藏
