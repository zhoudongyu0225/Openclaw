# 抖音脚本 V11：抑郁症的10个信号

## 脚本时长：30秒

### [0-3s] 数据切入
画面：年轻人低头，情绪低落
配音：全球3.5亿人抑郁，但80%的人不知道自己有病。

### [3-15s] 10个信号
画面：快速切换各种场景
配音：对照10个信号：
持续情绪低落、对所有事情失去兴趣、失眠或嗜睡、
食欲改变、疲惫乏力、注意力不集中、自我否定、
悲观绝望、身体查不出问题、行为改变。

### [15-25s] 严重程度判断
画面：医生出镜
配音：占3条以下，可能是暂时心情不好。
4-6条，可能是轻度抑郁。
7条以上，必须就医。

### [25-30s] 引导
画面：眠心诊所 Logo
配音：抑郁症不是想不开，而是一种可以治好的病。
回复"评估"，医生帮你分析。
