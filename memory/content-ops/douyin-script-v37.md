# 抖音脚本 V37：咖啡因影响睡眠（痛点共鸣型）

【开场】
（画面：凌晨3点主人公睁着眼看天花板）
"为什么喝完咖啡睡不着？"

【揭示真相】
"不是咖啡让你兴奋，
是咖啡因阻止了'困倦信号'传达到大脑。
你其实很困，但大脑收不到。"

【科学数据】
"咖啡因的半衰期是5-6小时。
下午2点喝一杯咖啡，
晚上8点还有一半在身体里，
凌晨1点仍有1/4影响你睡觉。"

【深度睡眠危害】
"更可怕的是，咖啡因会缩短深度睡眠。
深度睡眠是身体修复的时间，
即使睡8小时，深度睡眠不够，
第二天还是累。"

【常见误区】
"很多人以为'我喝咖啡不影响睡眠'，
其实只是习惯了这种低质量睡眠，
以为'睡醒累'是正常的。"

【解决方案】
"记住：
下午2点后不喝咖啡
睡前6小时是红线
用绿茶代替奶茶"

【结尾】
"转发给那个天天喝奶茶还问为什么睡不着的闺蜜"
