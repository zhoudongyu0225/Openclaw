# 抖音脚本 V16：抑郁症的10个信号

## 知识科普型

### 开头（0-3秒）
画面：灰暗滤镜，一个人坐在角落
文案：抑郁不是心情不好，而是生病了

### 内容（25-35秒）
1. **持续低落**
   "每天都不开心，对任何事都提不起兴趣"
   
2. **失眠或嗜睡**
   "要么睡不着，要么睡不醒"
   
3. **食欲改变**
   "要么吃不下，要么暴饮暴食"
   
4. **疲惫感**
   "什么都不想做，感觉身体被掏空"
   
5. **自我否定**
   "觉得自己没用，拖累别人"
   
6. **注意力无法集中**
   "看书、看电视都看不进去"
   
7. **对未来没有希望**
   "觉得生活没有意义"
   
8. **情绪波动大**
   "容易发脾气，或者突然崩溃"
   
9. **身体疼痛**
   "头痛、胸闷、这里疼那里疼"
   
10. **有轻生念头**
    "觉得活着没意思"

### 结尾（3秒）
画面：阳光照进房间
文案：中5条以上，建议就医。转发救人一命

### 时长：约40秒
### 风格：知识科普 + 关爱向
### 配乐：温情/励志
