# 抖音脚本 V6：3个毁掉睡眠的致命习惯

## 脚本时长：30秒

### [0-3s] 痛点切入
画面：凌晨2点，用户看着天花板，崩溃抓头
配音：凌晨2点了，脑子还是清醒的？明天还要上班，焦虑死了！

### [3-15s] 三个致命习惯
画面：快速切换三个场景
配音：毁掉你睡眠的，可能是这三个习惯：
第一，睡前刷手机，蓝光刺激大脑，根本睡不着。
第二，睡前想事情，焦虑激素飙升，越想越精神。
第三，强迫自己睡，越努力越清醒。

### [15-25s] 正确方法
画面：轻松的睡前场景（看书、冥想）
配音：正确的做法是：放下手机，让大脑自然放空。
睡不着就躺着，别焦虑，身体会自动修复。

### [25-30s] 引导行动
画面：眠心诊所 Logo + 引导
配音：长期失眠、靠自己也睡不着的话，回复"评估"，医生帮你分析问题。

## 备注
- 语速：稍快，带点焦虑感
- BGM：轻快的节奏，转折处可以有音效
- 字幕：关键词要突出
- 目标：完播率 >40%，收藏 >200
