# 抖音脚本 V22：睡眠质量的5个等级

## 类型：趣味科普型

### 开头（3秒）
"你睡得好不好？"
"先别急着回答。"
"看看你在第几级。"

### 内容（30秒）
"睡眠质量分5个等级："

"第5级：沾床就睡，一觉到天亮
闹钟响之前自然醒
这是顶级睡眠，非常健康"

"第4级：需要20分钟以内入睡
偶尔做梦，但醒来记得
睡眠质量不错"

"第3级：需要30分钟以上才能入睡
多梦，醒来记得内容
有点睡眠问题，需要注意"

"第2级：频繁夜醒，醒来很难再睡
白天犯困，记忆力下降
已经是睡眠障碍"

"第1级：整晚睡不着，或者只能浅睡
身体吃不消，情绪也崩溃
必须就医"

### 引导（7秒）
"你在哪一级？"
"3级以下，该重视了。"

### 结尾（5秒）
"关注我，每天教你睡好觉。"
