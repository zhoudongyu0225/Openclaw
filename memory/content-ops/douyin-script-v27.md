# 抖音脚本 V27：睡前焦虑自救指南

**类型**：知识干货型  
**时长**：45-60秒  
**目标人群**：20-40岁压力大、容易焦虑失眠的人群

---

## 开头（3秒）
画面：昏暗房间，只有一盏小台灯
台词：/"很多人问我，睡前一直焦虑睡不着怎么办？今天告诉你4个方法/"


## 干货内容（40秒）

### 方法1：焦虑具体化（10秒）
画面：手机屏幕亮着，备忘录界面
台词：/"第一，把焦虑写下来。不要让它在你脑子里转圈，写下来，你才能处理它。/

### 方法2：478呼吸法（10秒）
画面：安静的卧室，人躺着做呼吸
台词：/"第二，478呼吸法。吸气4秒，屏住7秒，呼气8秒。重复3次，副交感神经就会被激活。/

### 方法3：478呼吸法（10秒）
画面：冷战手机时间，锁屏显示23:00
台词：/"第三，睡前1小时手机强制离手。不是不看，是提前看，让大脑有时间冷却。/

### 方法4：身体扫描（10秒）
画面：人躺在床上，闭眼，从头到脚放松
台词：/"第四，身体扫描。从头到脚，一个部位一个部位地放松。你会发现，不知道什么时候就睡着了。/


## 结尾（5秒）
画面：Easy face + 诊所logo
台词：/"睡眠是最好的药。关注我，教你睡个好觉。/


## 字幕设计
- 关键数字放大：4秒、7秒、8秒、1小时
- 方法名称加粗标注
- 轻松配乐：钢琴轻音乐

## 拍摄建议
- 色调偏暖、暗，氛围感
- 主播声音低沉、温柔
- 多用第一人称讲述，增加信任感
