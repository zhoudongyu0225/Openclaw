# 抖音脚本 V21：睡前焦虑自救指南

## 类型：知识干货型

### 开头（3秒）
"睡前总是想太多，停不下来？"
"试试这个方法。"

### 痛点引入（5秒）
"脑子里像放电影：明天的工作、还没回的消息、银行卡余额..."
"越想越焦虑，越焦虑越睡不着。"
"其实，你不是一个人。"

### 干货教学（20秒）
"教你4个'接地气'的睡前焦虑自救法："

"1. 打电话给闺蜜/兄弟吐槽"
"把焦虑说出来，而不是憋在心里"

"2. 写下来"
"拿张纸，把担心的事全写下来，写完撕掉"

"3. 4-7-8呼吸法"
"吸气4秒，屏住呼吸7秒，呼气8秒，重复3次"

"4. 允许自己失眠"
"越是告诉自己'必须睡着'，越睡不着。不如放松，反而睡着了"

### 引导（7秒）
"收藏起来，今晚试试。"
"长期失眠焦虑，可以找我们聊聊。"

### 结尾（5秒）
"你睡前一般想什么？评论区聊聊。"
