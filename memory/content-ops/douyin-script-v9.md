# 抖音脚本 V9：什么情况下该去看心理医生？

## 脚本时长：30秒

### [0-3s] 痛点切入
画面：年轻人拖着疲惫的身体，坐在路边
配音：是不是经常觉得累？但又睡不着、吃不下、高兴不起来？

### [3-12s] 5个自检标准
画面：快速列出5个手指
配音：对照这5个标准，看看自己要不要看医生：
第一，症状超过一个月
第二，已经影响到工作学习
第三，感到极度痛苦
第四，自己调节了没用
第五，身体不舒服但查不出原因

### [12-22s] 误区澄清
画面：医生出镜，摆手
配音：很多人不敢看心理医生，怕被当作精神病。
其实就像感冒要看医生一样，心理感冒了也要看。

### [22-30s] 引导行动
画面：眠心诊所 Logo + 评估按钮
配音：不知道自己要不要看？回复"评估"，医生帮你判断。

---

## 备注
- 语速：适中，清晰
- BGM：轻快但有力量
- 字幕：5个标准要清晰列出
- 目标：完播率 >35%，评论互动 >100
