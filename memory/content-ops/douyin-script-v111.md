# V111：睡醒口干原因

**类型**：问题解决型

**脚本**：

你有没有这种感觉：

早上醒来，嘴巴干得像沙漠？

这可能不是小事，可能是身体在提醒你。

第一个原因：用嘴呼吸。

睡觉时鼻塞或者打呼噜，会让你不自觉用嘴呼吸。

口腔水分蒸发，第二天起来就口干。

第二个原因：药物副作用。

有些降压药、抗抑郁药会导致口干。

第三个原因：室内太干。

暖气、空调会让房间湿度下降。

怎么办？

用加湿器，保持卧室湿度40-60%。

睡前适量喝点水。

治疗鼻塞问题。

如果长期这样，最好去医院检查一下。
