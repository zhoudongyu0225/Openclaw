# V129：失眠别做这4件事（知识科普型）

## 【开场】
"越强迫自己睡越清醒？你的失眠可能是被"治"出来的"

## 【正文】
"失眠后千万别做这4件事，会越来越严重：

第一，强迫入睡。告诉自己赶紧睡，结果越想越清醒，大脑形成条件反射，一上床就焦虑。

第二，反复看时间。'已经2点了怎么办'——焦虑就是这样升级的。

第三，白天补觉。昨晚没睡好白天狂睡，晚上更睡不着，恶性循环。

第四，喝酒助眠。确实能快速入睡，但会破坏后半夜睡眠，容易醒，醒 了就再也睡不着。

正确的做法是：
睡不着就起来，不看时间，白天不补觉，睡前4小时不喝酒。"
