# 抖音脚本V48：叫不醒可能是病

## 类型：问题解决型
## 时长：30秒

【画面】
男主早上8个闹钟，全部没听见
字幕：每天睡12小时还是困？

【配音】
"有人每天睡够8小时还是很困，有人站着都能睡着。"

【画面】
男主在办公室突然倒下睡着，同事吓到
字幕：这不是懒，是病

【配音】
"别笑，这可能不是懒，是发作性睡病。"

【画面】
科普：发作性睡病四大症状
字幕：白天突然入睡、情绪激动时猝倒

【配音】
"发作性睡病有四大症状：白天突然无法控制地入睡，情绪一激动就全身发软倒地，睡前醒后出现幻觉，或者鬼压床。"

【画面】
对比表格
字幕：普通嗜睡vs发作性睡病

【配音】
"普通嗜睡是因为熬夜，补觉能好。但发作性睡病是神经系统问题，你控制不了。"

【画面】
男主去医院检查
字幕：怀疑自己有问题？去医院睡眠中心

【配音】
"如果你觉得自己有这个问题，去医院睡眠中心做个检查。"

【结尾】
字幕：转发给那个"睡神"朋友
