# 抖音脚本 V31：睡前吃这些=失眠

## 类型：知识科普型

## 时长：30-45秒

## 开头（3秒）
画面：深夜一个人在吃火锅、喝咖啡、吃蛋糕
配音：睡前吃这些食物 = 失眠

## 科普时间（25秒）
画面：每说一个食物切换一个画面
配音：第一名，咖啡和浓茶——咖啡因让你清醒到凌晨
第二名，酒精——让你快速入睡，但让你整晚醒
第三名，辛辣食物——刺激肠胃，大脑兴奋
第四名，高糖甜食——血糖波动，半夜突然醒
第五名，加工肉类——难以消化，负担太重

## 结尾（5秒）
画面：推荐食物
配音：那吃什么能助眠？温牛奶、坚果、香蕉。关注我，帮你睡个好觉。

## 字幕设计
- 食物名称用大字
- 配合对应食物画面
- 节奏要快
