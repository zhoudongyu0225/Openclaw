# 抖音脚本 V14：睡前不要做这5件事

## 痛点共鸣型

### 开头（0-3秒）
画面：深夜，主人公躺在床上辗转反侧
文案：又失眠了？可能因为你睡前做了这5件事

### 内容（15-30秒）
1. **刷短视频**
   "短视频会让大脑兴奋，越刷越清醒"
   
2. **想太多**
   "睡前焦虑等于给大脑加班，它也需要休息"
   
3. **喝酒**
   "酒精会破坏睡眠结构，半夜容易醒"
   
4. **喝咖啡**
   "咖啡因半衰期6小时，下午3点喝都可能睡不着"
   
5. **开灯睡觉**
   "光线会抑制褪黑素分泌，影响睡眠质量"

### 结尾（3秒）
画面：主人公睡着，画面出现"关注我，每天教你睡好觉"
文案：转发给那个睡不着的TA

### 时长：约30秒
### 风格：痛点共鸣 + 实用干货
### 配乐：轻音乐
