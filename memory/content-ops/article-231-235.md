## 文章231：睡前这个水果别吃，越吃越清醒

有些水果睡前吃会影响睡眠，但可能出乎你意料。

### 影响睡眠的水果

1. **香蕉**
   - 富含维生素B6
   - 会刺激神经系统
   - 少量食用

2. **苹果**
   - 纤维丰富
   - 胃肠道活跃
   - 影响入睡

3. **葡萄**
   - 含糖量高
   - 能量释放慢
   - 影响深度睡眠

4. **橙子**
   - 酸性较强
   - 刺激胃酸
   - 影响睡眠

### 适合睡前吃的水果

1. **樱桃**
   - 含褪黑素
   - 帮助入睡

2. **猕猴桃**
   - 维生素C丰富
   - 改善睡眠质量

3. **草莓**
   - 温和不刺激
   - 适量食用

### 吃水果的最佳时间

- 睡前2小时完成
- 不要吃得太多
- 避免高糖水果

## 文章232：为什么有人开车就犯困？可能是这个原因

开车犯困不是小事，可能危及生命。

### 开车犯困的原因

1. **睡眠不足**
   - 前一晚没睡好
   - 长期睡眠债务
   - 睡眠质量差

2. **道路环境**
   - 高速公路单调
   - 视线长时间固定
   - 缺氧

3. **车辆环境**
   - 车内温度过高
   - 二氧化碳积聚
   - 噪音单调

4. **生理因素**
   - 餐后血糖波动
   - 服用药物
   - 疾病影响

### 危险信号

- 频繁眨眼
- 揉眼睛
- 打哈欠
- 注意力不集中
- 偏离车道

### 解决方法

1. 保证充足睡眠
2. 定时休息（每2小时）
3. 开窗通风
4. 喝咖啡（见效后休息）
5. 听动感音乐

## 文章233：深度睡眠不够？可能是因为你吃了这些药

很多常用药物正在悄悄偷走你的深度睡眠。

### 影响睡眠的药物

1. **降压药**
   - β受体阻滞剂
   - 利尿剂
   - 影响睡眠结构

2. **抗抑郁药**
   - SSRI类
   - 失眠常见副作用
   - 早期更明显

3. **激素类**
   - 甲状腺激素
   - 糖皮质激素
   - 口服避孕药

4. **其他药物**
   - 抗生素
   - 止痛药
   - 抗过敏药

### 药物影响睡眠的表现

- 入睡困难
- 浅睡眠增多
- 频繁醒来
- 噩梦增多

### 应对方法

1. 咨询医生是否可以调整用药时间
2. 避免睡前服用
3. 改善睡眠习惯
4. 必要时使用助眠产品

### 重要提醒

不要自行停药！任何药物调整都需要医生指导。

## 文章234：睡觉时身体抖动是病吗？专家告诉你真相

睡眠中身体抖动是常见现象，有些是正常的，有些需要警惕。

### 生理性肌阵挛

1. **睡眠启动肌阵挛**
   - 入睡时肢体突然抖动
   - 感觉像"坠落"
   - 正常生理现象

2. **夜间肌阵挛**
   - 腿部抽动
   - 每20-40秒一次
   - 影响睡眠质量

### 病理性抖动

1. **不宁腿综合征**
   - 腿部不适感
   - 活动后缓解
   - 夜间加重

2. **周期性肢体运动障碍**
   - 规律性抽动
   - 无法控制
   - 需要治疗

3. **癫痫**
   - 全身抽搐
   - 伴随其他症状
   - 需要就医

### 如何区分

- 偶尔抖动：正常
- 频繁抖动：就医
- 伴随其他症状：警惕

### 改善方法

1. 规律作息
2. 减少咖啡因
3. 适度运动
4. 补充镁元素

## 文章235：你的睡眠日记应该这样写

睡眠日记是诊断睡眠问题的重要工具，正确记录有助于发现问题。

### 睡眠日记内容

1. **睡眠时间**
   - 上床时间
   - 入睡时间
   - 起床时间
   - 总睡眠时长

2. **睡眠质量**
   - 醒来次数
   - 再次入睡时间
   - 睡眠质量评分

3. **日间状态**
   - 午睡时间
   - 疲劳程度
   - 情绪状态

4. **生活习惯**
   - 咖啡因摄入
   - 运动情况
   - 饮酒

### 记录模板

| 日期 | 上床 | 入睡 | 起床 | 睡眠时长 | 夜间醒来 | 午睡 | 咖啡因 | 运动 | 质量评分 |
|------|------|------|------|----------|----------|------|--------|------|----------|

### 记录要点

1. 每天记录
2. 客观准确
3. 记录2周以上
4. 携带就医

### 分析方法

- 平均睡眠时长
- 入睡时间是否规律
- 睡眠效率计算
- 找出影响因素
