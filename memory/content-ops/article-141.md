# 公众号文章 140-143

## 文章141：为什么开会时总是忍不住犯困？

你有没有这种感觉：一进会议室就开始犯困，尤其是下午的会议，简直是"催眠现场"？

这真的不是你的问题。

### 会议室是天然的"睡眠舱"

1. **光线昏暗**
   大多数会议室为了投影或屏幕显示，会拉上窗帘、调暗灯光——这正是大脑分泌褪黑素的条件。

2. **空气不流通**
   密闭空间二氧化碳浓度升高，让人昏昏欲睡。

3. **声音单调**
   同事的声音、领导的讲话...这种单一的音频输入最容易让人走神犯困。

4. **坐姿固定**
   长时间坐在舒适的椅子上，身体血液循环变慢，大脑供氧减少。

### 还有一个重要原因：睡眠不足

如果你经常在开会时犯困，可能是身体在提醒你——昨晚的睡眠质量不够。

研究发现，睡眠不足7小时的人，在单调的会议环境中，犯困的概率增加40%。

### 如何在开会时保持清醒？

1. **坐直、稍微前倾**
   这个姿势能让呼吸更顺畅，大脑供氧更充足。

2. **记笔记**
   动手写的过程让大脑保持活跃。

3. **适时喝水**
   小口喝水保持口腔湿润，也能提神。

4. **坐靠近发言者的位置**
   参与度越高，越不容易犯困。

5. **提前补觉**
   如果知道下午有重要会议，中午适当小睡15-20分钟。

---

如果开会犯困已经成为常态，那该反思的不是你的专注力，而是你的睡眠。真正的解决方案，是在前一晚睡个好觉。
