# V112：手环监测准确吗

**类型**：知识科普型

**脚本**：

现在很多人戴着手环测睡眠。

但我今天要泼盆冷水：

手环测睡眠，其实没那么准。

它靠什么？加速度计、心率传感器。

通过算法推测你睡得好不好。

问题是：它能测出你几点睡、几点醒。

但它分不清你到底有没有深度睡眠。

研究显示：手环对睡眠时间的判断还可以。

但对睡眠质量的判断，经常不准。

深睡可能被误判，浅睡也可能被当成深睡。

那什么情况需要专业监测？

打呼噜严重、白天总是犯困、怀疑自己有睡眠呼吸暂停。

这种情况，去医院做脑电图监测，更靠谱。
