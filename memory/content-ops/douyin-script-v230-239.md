# 抖音脚本 V230-V239

## V230：说梦话是病吗（知识科普型）

【画面】
博主坐在床上，假装说梦话："老板，这个方案不行，重做！"

【配音】
睡觉说梦话，很多人觉得是睡得香。其实可能是身体在报警！

【字幕】
说梦话可能暗示的问题：
① 压力太大
② 睡眠呼吸暂停
③ 神经系统疾病

【配音】
如果偶尔说没问题，但如果经常说，还伴有白天犯困、记忆力下降，去医院看看。

【结尾】
关注我，带你睡个好觉。
