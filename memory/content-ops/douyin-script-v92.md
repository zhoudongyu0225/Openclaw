# 抖音脚本 V92：打呼噜伤心脏

## 类型：问题解决型

## 脚本

【画面：老公打呼噜，老婆被吵醒，崩溃】

别再以为打呼噜是"睡得香"了！

有一种打呼噜——呼吸暂停——正在偷偷伤害你的心脏。

【危险信号】

你的呼噜如果有这几种特征，很危险：
① 声音大到关门都能听见
② 打着打着突然停了，半天出一口气
③ 早上起来头晕头痛
④ 白天困得不行，随时能睡着

【危害讲解】

每次呼吸暂停，大脑就缺氧一次。长期这样：
- 血压越来越高
- 心脏负担越来越重
- 冠心病、心律失常风险翻倍

【怎么办】

4个方法：
1. 先去医院做个睡眠监测
2. 减重，肥胖是主要原因
3. 侧睡，侧卧能减轻阻塞
4. 戒烟酒，酒精会让喉部肌肉更松弛

【结尾】

打呼噜不是小事转发给身边那个"打呼噜"的他。

我是眠心，帮你睡个好觉。
