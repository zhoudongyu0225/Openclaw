# 睡眠日记怎么写？90%的人方法错了

---

"我昨天失眠了。"

"我最近睡眠很差。"

如果你只能说出这两句话，说明你的睡眠问题，永远找不到根源。

**睡眠日记，是解决失眠的第一步。**

## 为什么医生都让你写睡眠日记？

因为——

- 你的记忆会骗人，但数据不会
- 睡眠问题往往是"组合拳"，需要一个一个拆解
- 只有找到规律，才能对症下药

## 睡眠日记正确写法（附模板）

### 基础信息栏
| 项目 | 内容 |
|------|------|
| 日期 | 2026-03-08 |
| 星期 | 周日 |

### 睡前状态（22:00-躺床前）
| 问题 | 记录 |
|------|------|
| 几点吃晚饭？ | 19:00 |
| 睡前2小时吃什么了？ | 喝了咖啡 |
| 运动了吗？几点？ | 21:00跑步 |
| 情绪状态？ | 焦虑（工作的事） |
| 睡前手机看了多久？ | 40分钟 |

### 睡眠情况（躺床后）
| 问题 | 记录 |
|------|------|
| 几点上床？ | 22:30 |
| 几点入睡？ | 23:30 |
| 几点醒来？ | 2:30 |
| 醒来几次？ | 2次 |
| 最后几点醒的？ | 4:00，再也睡不着 |
| 做梦了吗？ | 是，很多梦 |

### 晨起状态
| 问题 | 记录 |
|------|------|
| 几点起床？ | 6:30 |
| 起床后状态？ | 累，头晕 |
| 白天困不困？ | 非常困 |

## 连续记录2周，你会看到

坚持写2周睡眠日记，你会发现——

原来你每次失眠，都是因为：
- 喝了咖啡 ☕
- 睡前看了手机 📱
- 或者那天发生了什么事 😟

**这就是规律。找到规律，就找到了解决方法。**
