# 抖音脚本 V30：睡前千万不要这样运动

## 类型：痛点共鸣型

## 时长：30-45秒

## 开头（3秒）
画面：昏暗房间，一个人在垫子上疯狂做运动
配音：你有没有发现？运动完反而更睡不着了？

## 痛点共鸣（10秒）
画面：切换到各种错误运动方式
配音：很多人觉得，累了就能睡。但其实——睡前剧烈运动，只会让你的交感神经更兴奋，体温降不下来，大脑清醒得一批！

## 干货输出（20秒）
画面：出现文字tips
配音：记住这三点：
第一，睡前3小时别运动，让身体有时间降温
第二，别做太剧烈的，选中等强度，微微出汗就行
第三，最好的运动时间是早上或下午

## 结尾（5秒）
画面：出诊logo和slogan
配音：运动方式不对，越运动越清醒。关注我，科学睡眠，从正确运动开始。

## 字幕要点
- 关键词用大字突出：睡前3小时、中等强度、早上/下午
- 配合节奏点切换画面
