# 抖音脚本 V24：你为什么总是睡不好？可能是这个原因

## 类型：痛点共鸣型

### 开头（3秒）
"你是不是也这样？"
（画面：凌晨辗转反侧的人）

### 内容（25秒）
"每天晚上10点就上床，
但就是睡不着。"
"脑子里像放电影，
停不下来。"

"越睡不着越焦虑，
越焦虑越睡不着。"
"恶性循环。"

"其实你睡不着的原因，
可能只有一个："
"你的交感神经，太兴奋了。"

### 科普（12秒）
"白天工作用的是交感神经，
晚上需要切换到副交感神经。"
"但很多人不会切换。"
"这就是为什么你睡前总想事，
睡眠浅，容易醒。"

### 引导（5秒）
"想知道怎么切换吗？"
"下期教你。"

### 结尾（5秒）
"关注我，教你睡个好觉。"
