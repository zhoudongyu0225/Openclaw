# V42：你的睡姿可能一直是错的

## 类型：知识科普型
## 时长：30-45秒

### 画面设计

**开场（0-3秒）**
画面：各种奇葩睡姿混剪——大字型、蜷缩型、掉床边型
配音：睡姿不对，越睡越累！今天告诉你正确答案

**第二幕（3-12秒）**
画面：三个人形分别展示仰卧、左侧卧、右侧卧
配音：三种睡姿优缺点——仰卧最舒服但容易打呼；左侧卧不压心脏还助消化；右侧卧最适合大部分人

**痛点（12-20秒）**
画面：睡醒落枕、腰痛、肩酸的表情包
配音：如果你经常落枕腰酸肩膀痛，先别怪枕头，看看你的睡姿

**实用（20-35秒）**
画面：字幕+真人演示
配音：记住一个原则——要让脊柱在一条直线上。侧卧时在两腿之间夹个枕头，仰卧时膝下垫个枕头

**结尾（35-45秒）**
画面：换好姿势安然入睡
配音：转发给那个睡姿奇葩的TA～

### 字幕要点
- 三种睡姿用图标+文字标注
- "脊柱在一条直线"加粗强调
- 结尾互动"转发给TA"
