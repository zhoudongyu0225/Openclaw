# 抖音脚本 V10：心理治疗的5个阶段

## 脚本时长：45秒

### [0-5s] 提问切入
画面：用户疑惑脸
配音：心理治疗到底在做什么？今天带你了解全过程。

### [5-15s] 第一阶段：评估
画面：医生在问诊，患者在回答
配音：第一步，全面评估。医生会了解你的症状、病史、生活环境，
做心理测评。这是最关键的一步。

### [15-25s] 第二阶段：制定方案
画面：医生在写治疗计划
配音：第二步，制定方案。确定治疗目标、方法、周期。
在眠心诊所，医生会告诉你什么时候能好。

### [25-35s] 第三阶段：系统治疗
画面：多种治疗方式（药物、心理、物理）
配音：第三步，系统治疗。不是简单的开药或聊天，
而是药物+心理+物理，综合施策。

### [35-42s] 第四阶段：巩固
画面：患者微笑，准备离开
配音：第四步，巩固维持。症状消失后，还要巩固1-2个月，防止复发。

### [42-45s] 引导
画面：眠心诊所 Logo
配音：心理治疗不是玄学，而是一个科学的过程。回复"评估"，了解你的治疗方案。

---

## 备注
- 风格：知识科普型
- 语速：适中
- 画面：简洁、专业
- 目标：收藏 >200，转发 >100
