# 抖音脚本V45：8小时不够可能是这个问题

## 类型：问题解决型
## 时长：35秒

【画面】
男主早上起床，8个闹钟都定了，还是起不来
字幕：睡够8小时还是很累？

【配音】
"为什么你睡够8小时还是累？不是你懒，可能是睡眠周期没对齐！"

【画面】
科普动画：90分钟一个圈
字幕：睡眠周期=90分钟

【配音】
"人的睡眠是由90分钟一个周期组成的，包括浅睡、深睡、REM做梦。"

【画面】
闹钟在深睡期响起，红色的"头晕"
字幕：周期中间被叫醒=睡眠惯性

【配音】
"如果你在深睡期被闹钟叫醒，就会出现睡眠惯性——头晕、反应慢、累成狗。"

【画面】
给出公式
字幕：起床时间-90分钟×周期数-15分钟入睡时间

【配音】
"正确做法：按90分钟倍数倒推入睡时间。比如7点起床，倒推5个半小时，就是该睡觉的时候。"

【画面】
男主定好闹钟，安稳入睡
字幕：对齐周期，比堆时间更重要

【配音】
"别再执着8小时了，对齐周期才能真正睡好。"

【结尾】
字幕：关注我，每天教你睡个好觉
