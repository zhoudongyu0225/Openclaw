# 抖音脚本V44：睡前这杯水别喝

## 类型：生活妙招型
## 时长：25秒

【画面】
女主睡前床头放了一杯水，吨吨吨喝完
字幕：睡前吨吨吨

【配音】
"睡前喝太多水的人，恭喜你，成功毁掉了自己的睡眠！"

【画面】
演示起夜流程
字幕：起来→开灯→去厕所→回来→睡不着

【配音】
"一次起夜=一次睡眠中断。大脑一旦被唤醒，想回深度睡眠？难！"

【画面】
出现数据
字幕：一次起夜=熬夜4小时

【配音】
"研究说，一次起夜，第二天的疲劳感相当于熬夜4小时。"

【画面】
给出建议
字幕：睡前2小时不喝水

【配音】
"正确做法：睡前2小时别喝，晚上渴了只小口润喉。"

【结尾】
字幕：转发给那个睡前必喝水的人
