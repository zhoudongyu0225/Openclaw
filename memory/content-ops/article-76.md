# 文章76：睡姿不对，越睡越累！哪种睡姿最适合你？

人这一辈子，三分之一的时间都在睡觉。

但你知道吗？睡姿不对，越睡越累。错误的睡姿不仅影响睡眠质量，还可能加重身体问题。

今天我们来聊聊：你的睡姿真的睡对了吗？

## 三种主要睡姿的利弊

### 仰卧睡姿

**优点：**
- 脊柱受力均匀，不容易落脸部不枕
- 接触枕头，减少皱纹
- 内脏不受压迫

**缺点：**
- 舌根容易后坠，打呼噜的人会加重
- 容易手麻（压到手臂神经）
- 仰卧时重力会让背部肌肉放松过度，醒后容易腰酸

**适合：** 颈椎病患者、腰椎间盘突出患者

### 左侧卧睡姿

**优点：**
- 减轻心脏压力（心脏在左侧）
- 有利于消化系统工作
- 孕妇首选（减轻子宫对血管的压迫）

**缺点：**
- 长期左侧卧可能压迫左耳，影响听力
- 左侧肩膀容易酸痛

**适合：** 孕妇、消化不良者、高血压患者

### 右侧卧睡姿

**优点：**
- 不压迫心脏
- 有利于肝脏解毒
- 减轻打呼噜

**缺点：**
- 长期右侧卧可能压迫右侧肺部
- 右肩容易酸痛

**适合：** 大部分成年人、肝脏有问题的人

## 根据身体问题选睡姿

| 身体问题 | 推荐睡姿 |
|---------|---------|
| 颈椎病 | 仰卧或侧卧（枕头要低） |
| 腰椎问题 | 仰卧（膝下垫枕头） |
| 高血压 | 侧卧（避免心脏受压） |
| 胃食管反流 | 左侧卧+抬高上半身 |
| 打呼噜 | 侧卧 |
| 孕妇 | 孕中后期左侧卧 |

## 特殊人群的睡姿指南

### 孕妇
孕中后期建议左侧卧，可以在两腿之间夹一个枕头，后背也可以垫一个。如果左侧卧不舒服，可以左右交替，但避免仰卧。

### 老年人
建议侧卧，尤其是右侧卧，减少心脏负担。如果有打呼噜或呼吸暂停问题，侧卧能明显改善。

### 颈椎病患者
仰卧时枕头高度约一个拳头（约10-15cm），要枕在颈椎而不是后脑勺。侧卧时枕头需要更高一些，保持颈椎与脊柱水平。

## 枕头高度有讲究

- 仰卧：一个拳头高度
- 侧卧：一个半拳头高度
- 枕头宽度：至少肩宽的1.5倍
- 材质：荞麦枕、乳胶枕、记忆枕都可以，主要看支撑性

---

你的最佳睡姿，是能让你在醒来时感到轻松的那个。

如果经常落枕、腰酸、肩膀痛，先别急着换床垫，先看看你的睡姿对不对。
