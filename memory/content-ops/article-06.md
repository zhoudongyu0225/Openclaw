## 文章6：正确的睡眠时间

### 标题：\"90%的人都睡错了！科学家告诉你最佳睡眠时间\"

---

**正文：**

\"我每天只睡5小时，是不是对身体不好？\"

\"睡太多会不会也不好？\"

\"几点睡几点起最健康？\"

关于睡眠时间，这是最全面的科学解答。

### 到底睡多久？

#### 各年龄段睡眠需求

| 年龄段 | 建议睡眠时间 |
|--------|--------------|
| 0-3个月 | 14-17小时 |
| 4-11个月 | 12-15小时 |
| 1-2岁 | 11-14小时 |
| 3-5岁 | 10-13小时 |
| 6-13岁 | 9-11小时 |
| 14-17岁 | 8-10小时 |
| 18-64岁 | 7-9小时 |
| 65岁以上 | 7-8小时 |

### 几点睡最好？

#### 传统说法：10点前睡

中医认为11点-1点是胆经排毒时间，应该入睡。

#### 科学说法：根据褪黑素分泌

褪黑素分泌高峰：晚上10点-凌晨2点

**结论：10点到11点之间入睡最好。**

### 熬夜能补回来吗？

**不能完全补回来。**

研究显示：
- 熬夜后补觉，身体状态只能恢复70%
- 连续熬夜1周，需要额外周末补觉4小时才能恢复
- 长期睡眠不足会导致不可逆的脑细胞损伤

### 睡眠质量比时间更重要

即使睡8小时，如果：
- 浅睡眠多
- 经常醒来
- 打呼噜（睡眠呼吸暂停）

也会影响健康。

### 正确的睡眠习惯

1. **固定作息**
   - 每天同一时间睡、同一时间起
   - 周末也不例外（±1小时以内）

2. **睡前仪式**
   - 睡前1小时不看手机
   - 睡前30分钟做放松活动

3. **环境**
   - 黑暗（伸手不见五指）
   - 安静（不超过40分贝）
   - 凉爽（18-22度）
