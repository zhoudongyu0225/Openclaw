# 文章135：今晚失眠怎么办？试试这5分钟急救入睡法

又是一个失眠的夜晚，盯着天花板，数了1000只羊还是清醒的？

别慌，今天教你5分钟急救入睡法。

## 1. 渐进式肌肉放松（2分钟）

从脚趾开始：
- 用力绷紧5秒
- 突然放松
- 感受血液回流的感觉

从脚到头，一个部位一个部位来。这是美国海军飞行员的御用方法，98%的人都能在2分钟内睡着。

## 2. 身体扫描冥想（1分钟）

闭眼，把注意力放在头顶：
- 想象一道温暖的光，从头顶慢慢往下移动
- 经过额头、眼睛、鼻子、嘴巴...
- 一直走到脚尖

不需要控制呼吸，只是"跟着光走"。

## 3. 4-7-8呼吸法（1分钟）

- 用鼻子吸气，数4下
- 屏住呼吸，数7下
- 用嘴缓慢呼气，数8下
- 重复3次

这个方法能直接触发副交感神经，让身体进入"休息模式"。

## 4. 心理解压（1分钟）

想一个让你安心的画面：
- 阳光下的海滩
- 童年夏夜的星空
- 猫咪在怀里打呼噜

不要想问题，不要想明天的事情，就让自己"单纯地呆着"。
