# V117：夜间醒来快速回睡（问题解决型）

【画面】
凌晨3点，主人公醒来，看手机。

【配音】
"凌晨醒来，看了一眼手机，然后失眠了？是不是你？"

【字幕】
"方法1：不要看时间"

【画面】
主人公把手机扣过去。

【配音】
"看时间 = 算还剩几小时 = 焦虑 = 失眠。把手机转过去。"

【字幕】
"方法2：放松身体"

【画面】
主人公伸展脚趾，放松肩膀。

【配音】
"别强迫自己'快睡'。先放松脚趾、肩膀、眼睛，身体放松了，大脑自然静。"

【字幕】
"方法3：478呼吸法"

【画面】
主人公做478呼吸。

【配音】
"吸气4秒，屏住7秒，呼气8秒。重复3次。直接触发副交感神经。"

【画面】
主人公安静地再次入睡。

【配音】
"夜间醒来是正常的，别把它当问题。越焦虑越睡不着。"

【字幕】
"转发给那个经常醒来的朋友"

【结束】
*眠心诊所*
