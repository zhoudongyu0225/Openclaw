# 文章137：为什么冬天睡更久还是困？季节性疲劳的真相

明明睡了10小时，冬天还是整天犯困？

你不是懒，你是被季节"调控"了。

## 1. 日照时间短

冬天白天短，阳光少。

阳光 = 维生素D = 能量。

阳光少 = 维生素D不足 = 疲劳。

## 2. 新陈代谢变慢

为了保温，身体会自动降低代谢。

就像手机开启"省电模式"，人也变得"省电"。

这在进化上是合理的，但现代人不需要冬眠啊！

## 3. 褪黑素分泌紊乱

天黑得早，褪黑素分泌提前。
但你可能还撑着不睡，这就乱了。

## 应对方法

1. **早起晒太阳**：哪怕只有10分钟
2. **补维生素D**：每天1000-2000IU
3. **室内开灯**：模拟白天光照
4. **适当运动**：让身体"醒过来"
5. **固定作息**：别因为天黑早就提前上床

---

冬天犯困是正常的，但如果你：

- 困到影响工作
- 情绪低落
- 体重增加

可能是季节性情绪障碍（ SAD ），建议看看医生。

转发给那个一到冬天就"昏迷"的朋友。
