# 抖音脚本 V25：女生为什么总失眠？

**类型**：痛点共鸣型
**时长**：45-60秒
**BGM**：轻音乐

---

### 画面1（0-5秒）
【画面】：女生躺在床上，盯着天花板
【配音】：为什么同样的压力，男生倒头就睡，你却睁眼到天亮？

### 画面2（5-20秒）
【画面】：动画/字幕展示
【配音】：第一个原因，激素在作祟。女生每个月雌激素和孕酮剧烈波动——经期前焦虑、孕期睡不好、更年期潮热。男生根本不用经历这些。

### 画面3（20-35秒）
【画面】：女生坐在床边思考
【配音】：第二个原因，情绪太发达。女生大脑天生对情绪更敏感，睡前容易复盘，容易替别人担心，一件小事能想出八百个结果。这叫反刍思维，是失眠的帮凶。

### 画面4（35-45秒）
【画面】：字幕列出
【配音】：第三个原因，家庭角色压力。中国女性承担的家务是男性的2.5倍，职场+家庭双倍压力，睡眠时间反而比男性少23分钟。

### 画面5（45-55秒）
【画面】：温暖色调，轻柔表达
【配音】：所以不是你不够努力，是身体天生更敏感。接受它，科学应对它。

### 画面6（55-60秒）
【画面】：眠心诊所logo+文字
【配音】：我是眠心诊所，帮你睡个好觉。
