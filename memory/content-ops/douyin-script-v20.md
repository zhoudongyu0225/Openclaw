# 抖音脚本 V20：测测你的抑郁程度

## 类型：互动测试型

### 开头（3秒）
"你以为你只是心情不好？"
"可能是身体在报警。"
"来做个测试。"

### 测评内容（25秒）
"过去两周，以下情况你出现过几次？"
"1. 对以前喜欢的事，完全提不起兴趣"
"2. 经常无缘无故想哭"
"3. 晚上失眠，或者早醒"
"4. 觉得自己很失败活着没意思"
"5. 容易发脾气，或者反应迟钝"
"每个问题：完全没有=0分，偶尔=1分，经常=2分，持续=3分"

### 结果解读（15秒）
"0-4分：正常情绪波动"
"5-9分：轻度抑郁倾向，注意调节"
"10-14分：中度抑郁，建议就医"
"15分以上：重度抑郁，必须重视"

### 引导（7秒）
"这个测试仅供参考，不等于诊断。"
"如果分数偏高，及时寻求专业帮助。"
"眠心诊所，专业心理诊疗。"

### 结尾（5秒）
"你多少分？评论区告诉我。"
"转发给那个最近不开心的朋友。"
