# 抖音脚本 V23：心理咨询师的真心话

## 类型：情感共鸣型

### 开头（3秒）
"作为心理咨询师，我想说几句心里话。"

### 内容（25秒）
"很多人问我："
"'去心理咨询，是不是说明我有病？'"

"不是的。"

"感冒了要吃药，心感冒了要找心理医生。"
"这不代表你脆弱，不代表你有问题。"
"只是你的心，需要被照顾一下。"

"来咨询的人，
有职场压力大的白领，
有婚姻遇到问题的夫妻，
有找不到方向的大学生，
也有只是想要更了解自己的人。"

"心理咨询，是给心灵做个SPA。"

### 引导（7秒）
"你对自己的心理状态满意吗？"
"不满意的话，可以来找我们聊聊。"

### 结尾（5秒）
"你因为什么问题纠结过？评论区说说。"
