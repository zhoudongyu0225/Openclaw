# 试玩广告HTML技术方案

## 概述
试玩广告让用户在落地页体验游戏核心玩法，提高转化率。

## 技术选型

### 方案1：原生Canvas（当前）
- 优点：轻量、无依赖
- 缺点：开发效率低

### 方案2：Phaser.js
- 优点：成熟游戏引擎
- 缺点：包较大

### 方案3：Three.js + 3D
- 优点：3D效果震撼
- 缺点：性能要求高

## 万龙觉醒试玩Demo功能

### 核心玩法
1. 摇杆控制移动
2. 普攻按钮
3. 3个技能按钮
4. 敌人AI
5. 血条系统
6. 资源系统

### 转化引导
- 30秒后弹出下载提示
- 可选：通关/升级后弹出

## 实现步骤

1. 搭建基础HTML结构
2. 实现Canvas渲染
3. 接入控制（摇杆+按钮）
4. 实现敌人AI
5. 添加音效
6. 优化性能
7. 添加转化弹窗
8. 测试各机型

## 性能优化

- 目标：3秒内加载完成
- Canvas分辨率：按设备调整
- 粒子数量限制
- requestAnimationFrame
- 对象池复用
