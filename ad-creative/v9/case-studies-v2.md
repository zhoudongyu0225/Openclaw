# SLG游戏爆款素材分析 v2

## 分析日期
2026-03-02

## 爆款素材类型

### 1. 试玩广告 (Playable Ads)
**特点**：让用户实际体验游戏核心玩法
- 开头3秒：抓住注意力（战斗/建造/升级）
- 中间：可交互的核心玩法
- 结尾：转化按钮 + 精美画面

**案例模板**：
```
【开头】英雄被包围 → 释放技能 → 敌人倒下
【交互】让用户建造/升级/释放技能
【结尾】展示更多内容 → 下载按钮
```

### 2. 剧情反转类
**特点**：制造冲突和反转
- 开场：展示游戏某个方面（战斗弱）
- 反转：原来是有策略/组合
- 结尾：爽快感 + 下载

**脚本模板**：
```
0-3秒：主角被打，求救
3-8秒：援军到达，阵型变换
8-15秒：反击，爽快感展示
15-20秒：下载引导
```

### 3. UE4高质量展示
**特点**：展示游戏品质
- 电影级画面
- 展示宏大场景
- 快速剪辑精彩片段

**制作要点**：
- 选用最高画质片段
- 配乐要震撼
- 节奏快（2-3秒一切换）

### 4. 社交场景类
**特点**：展示和朋友一起玩
- 联盟战
- 公会活动
- 多人副本

## 创意方向库

### 方向1：英雄成长
- 屌丝逆袭 → 变强 → 复仇
- 养成快感
- 装备/技能展示

### 方向2：策略深度
- 以弱胜强
- 阵型/搭配
- 脑洞策略

### 方向3：资源积累
- 收集快感
- 升级爽感
- 建筑养成

### 方向4：战斗爽感
- 技能连招
- 大招释放
- 割草快感

### 方向5：社交荣耀
- 联盟胜利
- 公会副本
- 排名展示
